# Volunteer Feedback Form - Questions

Use these questions to create a Google Form for collecting Phase 1 validation feedback.

**Form Title:** Herp Crossing - Volunteer Feedback

**Form Description:**
> Help us improve the amphibian patrol data collection system! Your feedback from real patrols is critical for Phase 1 validation. This takes 3-5 minutes.

---

## Section 1: About You

**Question 1: Observer Name**
- Type: Short answer
- Required: Yes
- Help text: Same name you use in the forms

**Question 2: How many patrols have you completed using these forms?**
- Type: Multiple choice
- Required: Yes
- Options:
  - 1 patrol
  - 2-3 patrols
  - 4-5 patrols
  - 6+ patrols

**Question 3: What device did you use for field data entry?**
- Type: Multiple choice
- Required: Yes
- Options:
  - iPhone (Safari)
  - Android phone (Chrome)
  - Android phone (other browser)
  - Tablet
  - Other: ___

## Analysis Plan

After collecting responses:

1. **Quantitative Analysis:**
   - Average time per form
   - Technical issue frequency
   - Feature usage rates
   - Satisfaction scores

2. **Qualitative Analysis:**
   - Common pain points
   - Feature requests
   - Workflow issues
   - Unexpected use cases

3. **Decision Making:**
   - Go/no-go on Phase 2
   - Required changes for Phase 2
   - Quick fixes for Phase 1 (if continuing)

---

**Google Form Setup Time:** ~15 minutes

**Response Collection Period:** Throughout Phase 1 testing (2-4 weeks)

**Target Responses:** At least 1 per volunteer, ideally after every 3-5 patrols

**Follow-up:** Personal interviews with 2-3 volunteers for deeper insights
